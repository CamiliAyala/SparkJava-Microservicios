# 🚀 Microservicios y MVC con Spark Java

Este repositorio contiene una colección de proyectos prácticos desarrollados para consolidar conceptos avanzados de arquitectura de software. El enfoque principal está en la construcción de aplicaciones web escalables utilizando el patrón **Modelo-Vista-Controlador (MVC)**, el desarrollo de servicios **REST**, y la persistencia de datos.

### 🛠️ Stack Tecnológico
* **Backend:** Java, Spark Web Framework.
* **Vistas (Frontend):** Apache Velocity (Motor de plantillas).
* **Persistencia & Base de Datos:** SQL2o (ORM ligero) y modelado relacional en SQL.

### 🌟 Implementaciones Destacadas
Dentro de los distintos módulos del repositorio se resuelven problemas reales de negocio, destacándose:

* **Motor de Recomendación (Estilo Netflix):** Modelado complejo de base de datos relacional (Películas, Directores, Reparto) que incluye un algoritmo para recomendar contenido basado en el historial de visualización de los usuarios.
* **Lógica de E-commerce y Logística:** Desarrollo de microservicios para cotizar rutas de transporte (origen/destino) y funcionalidades base para plataformas de pedidos.
* **Sistema de Fidelización (Canje de Puntos):** Implementación de reglas de negocio para validar y mostrar dinámicamente opciones de canje según el saldo de la cuenta del usuario.
